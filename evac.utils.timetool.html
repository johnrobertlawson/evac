Bases: object

A custom time object.

Example

To convert from a number of seconds to the largest units possible, suitable for a WRF namelist for instance, try the following:

secs = 10801
TT = TimeTool(seconds=secs)
d,h,m,s = TT.reduce()

There!

Note

  • Not for dates - for that, use datetime.datetime.
